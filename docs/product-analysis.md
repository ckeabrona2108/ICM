# ICM Product Analysis (based on SFEROOM public feature set)

## Core product modules

1. Onboarding and identity
- Registration and login
- Artist profile setup
- Subscription entry point

2. Distribution pipeline
- Create release
- Upload audio + cover
- Metadata and rights
- Submit to moderation
- Distribution tracking by platform

3. Operational dashboard
- Overview widgets
- Release statuses
- Recent moderation alerts
- Quick actions

4. Audience analytics
- Streams/listeners/saves
- Campaign impact metrics
- Release-level performance

5. Finance
- Balance and expected payouts
- Transaction history
- Payout requests
- Platform commission transparency

6. AI productivity
- Release description generation
- Press release generation
- Artist bio generation
- TikTok/Reels ideas
- Cover analysis and promotion recommendations

7. Growth and marketing
- Pitching workflows
- Ad campaign tracking
- Smart links and pre-save mechanics

8. Communication and support
- Messages
- Support tickets
- SLA/status visibility

9. Admin control plane
- User management
- Release moderation
- Financial oversight
- Internal admin logs

## ICM differentiation strategy

- Dashboard-first architecture with faster daily flow for artists/labels.
- Premium dark interface with clear hierarchy and reduced cognitive load.
- Better release wizard UX with strict validation and step-by-step progression.
- Unified AI + marketing loop: generation to campaign execution in one workspace.
- Backend-ready modularity (Prisma + API routes + mock domain layer).
